⚙ Configure o Client ID do Google para salvar no Drive

Abra este arquivo HTML em um editor de texto e substitua SEU_CLIENT_ID_AQUI.apps.googleusercontent.com pelo seu Client ID:

  1. Acesse console.cloud.google.com e crie um projeto
  2. APIs e Servicos → Biblioteca → ative Google Drive API
  3. Credenciais → Criar → ID do cliente OAuth 2.0 → Aplicativo da Web
  4. Em "Origens JS autorizadas" adicione: null (arquivo local) e https://drive.google.com
  5. Copie o ID do cliente e cole no inicio do arquivo HTML
